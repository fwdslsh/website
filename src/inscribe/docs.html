Command Reference

All commands and options

Learning Commands

learning add

inscribe learning add [OPTIONS]
FlagShortDescription
--type-tLearning type: pattern, problem-solution, mistake, insight
--summary-sBrief description (50-100 chars)
--content-cFull markdown content
--content-fileRead content from file
--domains-dKnowledge domains (comma-separated)
--toolsTools used (comma-separated)
--confidenceConfidence 0.0-1.0 (default: 0.5)
--sessionSource session ID
--output-oOverride output directory
--dry-runPreview without writing

learning edit/delete

inscribe learning edit 
inscribe learning delete

Session Commands

inscribe session start [OPTIONS]
inscribe session stop [OPTIONS]
FlagDescription
--nameSession identifier
--repoRepository path
--branchGit branch
--tagsSession tags (comma-separated)
--statusCompletion status (ok, failed, partial)

Event Commands

inscribe add   [OPTIONS]
FlagDescription
--json-fileRead event data from JSON file
--stdinRead from stdin
--tagsTags (comma-separated)

List Commands

inscribe list  [OPTIONS]

Types:
  learnings    List captured learnings
  sessions     List work sessions
  events       List all events
  decisions    List decisions
FlagDescription
--limitMaximum results to return
--tagsFilter by tags
--sinceFilter by date (YYYY-MM-DD)
--untilFilter by date (YYYY-MM-DD)

Hook Integration

# Platform hook integration
inscribe hook  --source  [OPTIONS]

# Examples:
cat hook-payload.json | inscribe hook post-tool-use --source claude-code
cat hook-payload.json | inscribe hook user-prompt-submit --source opencode

PostLearningCreated Hook

Emitted after a learning is written. Automatically triggers catalog-index hook.

Disable Hooks

export HYPHN_DISABLE_HOOKS=catalog-index
