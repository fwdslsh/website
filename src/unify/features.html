Why Unify?

Modern static sites with zero hassle.

Zero Learning Curve

Unify uses familiar include syntax (like <!--#include virtual="header.html" --> comments and <include src="header.html"> tags) for templating, so you can start building immediately without learning new languages or frameworks.

Modern Tooling

Under the hood, Unify is built with modern JavaScript and works on Node.js, Deno, and Bun. No heavy dependencies or complex bundlers – just a straightforward tool that fits into any workflow.

Live Reload & Development Server

A built-in development server provides live reload via Server-Sent Events. Your changes appear instantly in the browser, keeping your development flow smooth and helping you iterate faster.

Flexible Content Support

Write content in HTML or Markdown (with YAML frontmatter for metadata). Unify seamlessly processes both, wrapping Markdown content with your layouts. Static assets like images or CSS are also tracked and copied automatically when referenced.

SEO Optimized

Unify automatically generates a sitemap.xml and lets you define a universal head.html include for meta tags and scripts. This means better SEO and easy integration of analytics or social tags with minimal effort.

Framework-Free Output

The output of Unify is pure static HTML, CSS, and assets. There's no client-side framework needed, which means your site is lean, fast, and can be hosted on any server or CDN. Just build and deploy.
