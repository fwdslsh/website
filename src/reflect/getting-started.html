Getting Started

Usage tracking and learning analytics

Installation

# From monorepo root
npm install
# or
bun install

Basic Usage

Analyze Usage

# Analyze all usage events
reflect analyze usage

# Filter by time range
reflect analyze usage \
  --time-start 2025-01-01T00:00:00Z \
  --time-end 2025-01-31T23:59:59Z

Generate Reports

# Comprehensive report
reflect report comprehensive --output-path report.md

# Specific sections
reflect report comprehensive --sections usage,procedures

Environment Variables

VariableDefaultDescription
HYPHN_HISTORY_ROOT~/.hyphn/historyHistory logs directory
HYPHN_SESSIONS_ROOT~/.hyphn/sessionsSessions directory
HYPHN_INDEX_ROOT~/.hyphn/indexIndex directory

Configuration File

# ~/.reflectrc.json
{
  "historyRoot": "/home/user/.hyphn/history",
  "sessionsRoot": "/home/user/.hyphn/sessions",
  "indexRoot": "/home/user/.hyphn/index"
}
