A More Peaceful Way to Build Static Websites

Unify is a modern, lightweight static site generator that brings the power of server-side includes, markdown processing, and live reloading to your workflow. Build maintainable static sites with a component-based approach – no more copy-pasting headers and footers!

Get Started
npx @fwdslsh/unify serve

Key Features

  • Zero Learning Curve: Use standard HTML templates and familiar <include> tags or Apache SSI syntax for components.
  • Live Development: Built-in dev server with auto-reload for a smooth workflow.
  • Flexible & Fast: Supports Node.js, Deno, or Bun with lightning-fast builds and minimal dependencies.
  • SEO-Ready: Automatic sitemap generation and easy meta tag management.
