Team Collaboration

Create a shared git repository for your team's AI prompts and agents.

git clone git@github.com:my-org/ai-config.git ~/ai-config hyphn sources add ~/ai-config

Custom Agents

Define specialized agents for your specific codebase.

# ~/.hyphn/agents/db-expert.md --- name: db-expert description: Expert in our PostgreSQL schema --- You are a database expert. Always use parameterized queries. Refer to schema.sql for table definitions.
