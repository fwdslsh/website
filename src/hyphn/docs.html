CLI Reference

Usage: hyphn [command] [options]

Commands:
  init              Initialize ~/.hyphn directory
  setup             Configure integrations with installed tools
  list              List available agents, skills, and commands
  sources add       Add a new source repository
  sources rm        Remove a source repository
  sources update    Update all sources
  serve             Start MCP server

Configuration

Configuration is stored in `~/.hyphn/config.yaml`.
